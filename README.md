
# tic-tac-toe

This is just a project to learn a bit of rust


# Running

To play the game run
```bash
cargo run
```

you'll see
```
   |   |
___|___|___
   |   |
___|___|___
   |   |
   |   |

Player's X turn:
```

Put in the coordinates of your move. Eg: `1 1`
Then player O will be prompted for they're move.
```
   |   |
___|___|___
   | X |
___|___|___
   |   |
   |   |

Player's O turn:
```
